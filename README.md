## Practical Selection Checklist

- Confirm the platform supports the model you actually need, not just a marketing label.
- Test latency, context length, file upload, image generation, and conversation history before long-term use.
- Avoid uploading private keys, contracts, customer data, unreleased code, or personal identity documents to third-party mirror sites.
- Prefer platforms with clear pricing, support channels, refund rules, and stable domain history.
- Keep a backup option because AI model routing and regional access rules can change.

## FAQ

### Is this an official OpenAI repository?

No. This is an independent resource index. It is not affiliated with OpenAI, Anthropic, Google, or DeepSeek.

### Why use a mirror or aggregation platform?

For many China-based users, the official ChatGPT website has access, registration, payment, and account-risk barriers. Aggregation platforms can reduce those barriers by providing domestic access and local payment methods.

### Is the official ChatGPT Plus always better?

Not always. Official ChatGPT Plus is the cleanest option if you can access and pay for it reliably. Multi-model platforms can be more convenient if you need ChatGPT, Claude, Gemini, and DeepSeek in one place.
